# EdgeGestureHub 代码阅读顺序指南（代码对齐版）

## 主链路
`main.c -> App_Init/App_Loop -> IMU_ReadFrame -> Gesture_Run -> ControlLogic/FireWater/OLED`

## 阅读顺序
1. `Core/Src/main.c`
- 看外设初始化与 `App_Init/App_Loop` 接入。

2. `Core/Inc/app_config.h`
- 看调度周期与 NanoEdge 参数。

3. `Core/Src/app_task.c`
- 看 10/20/100ms 三段调度。
- 看双串口同时输出与 OLED 刷新。

4. `Core/Src/gesture_engine.c`
- 看 NanoEdge 初始化、学习阶段、检测阶段。
- 看 `cls` 映射：`0/1/2`。
- 看异常时阈值兜底分支。

5. `Core/Src/control_logic.c`
- 看 `ANOM` 边沿触发。
- 看 `500ms` 冷却与“回 STATIC 后再触发”门控。

6. `Drivers/BSP/Src/mpu6050.c`
- 看 I2C1 读数与单位换算。

7. `Drivers/BSP/Src/ssd1306.c`
- 看 I2C2 初始化、显存刷新、字符串显示。

## 快速事实（避免看错）
- MPU6050: `I2C1 PB6/PB7`
- OLED: `I2C2 PB10/PB11`
- USART1: `115200`
- USART2: `9600`
- FireWater: 纯 CSV `ax,ay,az,gx,gy,gz,cls`
- OLED 标签: `LEARN/NORM/ANOM`
- 业务输出: `PD12` 切换
