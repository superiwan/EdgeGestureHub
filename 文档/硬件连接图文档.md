### 电路模块说明

#### 1. 电源模块

```
        USB 5V
          │
          ├─── 100µF (C1) ─── GND  [电源滤波]
          │
          ├─── AMS1117-3.3 (U1)    [3.3V稳压]
          │      │
          │      ├─── IN  ───── 5V
          │      ├─── OUT ───── 3.3V
          │      └─── GND ───── GND
          │
          └─── 10µF (C2) ──── GND   [输出滤波]
                │
                └─── VCC_3.3V (供电给所有模块)
```

**元器件参数：**
- C1: 100µF/10V 电解电容（电源输入滤波）
- C2: 10µF/10V 钽电容（稳压输出滤波）
- U1: AMS1117-3.3 稳压芯片（800mA输出）

#### 2. I2C 总线电路

```
          VCC_3.3V
              │
              ├── 4.7KΩ (R1) ───┐
              │                 │
              │                 ├─── SDA ──── PB7 (STM32)
              │                 │
              ├── 4.7KΩ (R2) ───┤
              │                 │
              │                 ├─── SCL ──── PB6 (STM32)
              │                 │
              │                 │
    MPU6050 ──┴─────────────────┴─── SSD1306
```

**元器件参数：**
- R1/R2: 4.7KΩ 0805 电阻（I2C上拉）
- 上拉电压：3.3V
- 总线速率：400kHz

#### 3. MPU6050 传感器电路

```

                  ┌──────────────┐
```

**关键参数：**
- 工作电压：3.3V

- 采样率：20Hz
- 加速度范围：±4G

### 电路设计要点

#### 1. 电源设计
- ✅ 使用低 dropout 稳压器（AMS1117-3.3）
- ✅ 输入输出电容充足（100µF + 10µF）
- ✅ 每个IC就近放置去耦电容（100nF）
- ✅ 电源走线粗且短（≥20mil）

#### 2. I2C总线设计
- ✅ 使用4.7K上拉电阻到3.3V
- ✅ 总线走线等长匹配
- ✅ 避免长走线（<20cm）
- ✅ 远离高速信号线

#### 3. 信号完整性
- ✅ I2C信号线加地保护
- ✅ 串口线加限流电阻（1KΩ）
- ✅ 避免平行走线干扰
- ✅ 关键信号包地处理

#### 4. EMI/EMC设计
- ✅ 完整地平面
- ✅ 电源去耦充分
- ✅ 信号线阻抗控制
- ✅ 避免环路天线效应

### 调试与测试

#### 上电测试步骤
1. **电源测试**
   ```
   - 测量 VCC_3.3V 是否正常（3.25-3.35V）
   - 检查 GND 是否连通
   - 观察电源纹波（<50mV）
   ```

2. **I2C总线测试**
   ```
   - 扫描I2C设备（应检测到0x68和0x3C）
   - 测量SDA/SCL空闲电压（应≈3.3V）
   - 观察I2C波形是否正常
   ```

3. **传感器测试**
   ```
   - 读取加速度数据（应有变化）
   - 读取角速度数据（应有变化）
   - 读取温度数据（应接近室温）
   ```

4. **显示测试**
   ```
   - OLED初始化是否成功
   - 显示内容是否清晰
   - 刷新率是否正常
   ```

5. **串口测试**
   ```
   - 串口波特率是否匹配（115200）
   - 数据发送是否正常
   - VOFA+接收是否正常
   ```

## 🔧 I2C总线设计

### 总线拓扑

```
STM32F407 (Master)
    │
    ├── PB7 (SDA) ────┬──── MPU6050 (0x68)
    │                 │
    │                 └──── SSD1306 (0x3C)
    │
    └── PB6 (SCL) ────┬──── MPU6050
                      │
                      └──── SSD1306
```

### 上拉电阻配置

- **推荐值**：4.7kΩ
- **连接位置**：SDA和SCL各接一个上拉电阻到3.3V
- **作用**：保证I2C总线空闲时为高电平

|------|------|---------|

## ⚠️ 接线注意事项

### 1. 电源连接
- ✅ 所有设备GND必须共地
- ✅ MPU6050和OLED使用3.3V供电
- ✅ USB-TTL可提供5V供电（如需要）
- ⚠️ 确保电源供电能力足够（>500mA）

### 2. I2C连接
- ✅ MPU6050和OLED共用I2C总线
- ✅ 设备地址不同，不会冲突
- ⚠️ 建议使用4.7K上拉电阻
- ⚠️ I2C线长建议<20cm

### 3. 串口连接
- ✅ TX接RX，RX接TX（交叉连接）
- ✅ 波特率必须匹配（115200）
- ⚠️ 串口线长建议<1m

### 4. 布局建议
- 📍 MPU6050应靠近主控板
- 📍 I2C总线走线应短且直
- 📍 电源线应粗且短
- 📍 避免干扰源（电机、继电器等）

## 🛠️ 调试接口

### VOFA+ 配置

**协议设置：**
- 协议类型：JustFloat
- 数据格式：`AX:xx.xx AY:xx.xx AZ:xx.xx GX:xx.xx GY:xx.xx GZ:xx.xx Gesture:XXX`

**通道配置：**
1. AX：X轴加速度
2. AY：Y轴加速度
3. AZ：Z轴加速度
4. GX：X轴角速度
5. GY：Y轴角速度
6. GZ：Z轴角速度
7. Gesture：手势识别结果（字符串）

### 波形调试

**推荐调试步骤：**
1. 检查I2C设备是否正常枚举
2. 观察传感器数据波形是否平滑
3. 测试手势识别阈值是否合理
4. 优化滤波参数

## 🔍 常见问题

### Q1: I2C设备无响应
**解决方案：**
- 检查I2C地址是否正确（0x68/0x3C）
- 确认上拉电阻是否连接
- 使用I2C扫描工具检测设备

### Q2: OLED显示异常
**解决方案：**
- 检查供电电压（3.3V）
- 确认I2C地址（0x3C）
- 调整对比度设置

### Q3: 串口无数据
**解决方案：**
- 检查TX/RX是否交叉连接
- 确认波特率设置（115200）
- 检查USB-TTL驱动是否安装

### Q4: 传感器数据漂移
**解决方案：**
- 增加滤波算法
- 校准传感器零点
- 检查供电稳定性

## 🧩 天空星开发板适配结论（基于官方文档）

> 说明：你提供的 oshwhub 原始链接当前处于维护状态，无法直接读取原理图详情；以下结论基于立创开发板技术文档中心“天空星STM32开发板介绍/入门教程目录”整理。

### 1) 天空星板载通常已具备（无需重复增加）

1. MCU 最小系统（STM32F407VxT6核心）
2. 基础供电系统（板载3.3V电源链路）
3. 基础调试与下载通道（对应程序下载教程）
4. 基础板载资源（官方入门目录包含 LED、按键、ADC光敏、SPI Flash、RTC 等章节，说明板上提供或默认实验依托这些资源）

### 2) 针对本项目你仍需额外添加的模块

1. `MPU6050` 六轴传感器模块（I2C）
2. `0.96寸 SSD1306 OLED` 模块（I2C）
3. `USB-TTL` 串口模块（用于 VOFA+ 调试，若你改用板载虚拟串口可选）
4. I2C 上拉电阻 `4.7k x2`（仅当外接模块板上没有上拉时再加）
5. 跳线/连接器（面包板或扩展板连接）

### 3) 电阻电容是否还要加（天空星场景）

1. 不需要再给 MCU 主控重复加最小系统去耦与电源稳压（已在开发板上）
2. 对外接模块：
   - 若模块自带上拉/去耦，可不再额外加
   - 若模块未带上拉，需补 `SDA/SCL -> 4.7k -> 3.3V`
3. 串口 1k 保护电阻属于增强项，可按布线长度和抗干扰需求选加

### 4) 推荐最小外接 BOM（天空星版本）

| 序号 | 元器件 | 型号/规格 | 封装/形态 | 数量 | 是否必需 |
|---|---|---|---|---:|---|
| 1 | 六轴传感器模块 | MPU6050 | 模块 | 1 | 必需 |
| 2 | OLED显示模块 | SSD1306 0.96寸 I2C | 模块 | 1 | 必需 |
| 3 | 串口模块 | USB-TTL(CH340/CP2102) | 模块 | 1 | 可选 |
| 4 | I2C上拉电阻 | 4.7k | 0805/插件 | 2 | 条件必需 |
| 5 | 连接线 | 杜邦线 | 线材 | 若干 | 必需 |

### 5) 引脚建议（与本项目代码一致）

1. `PB6 -> I2C1_SCL`
2. `PB7 -> I2C1_SDA`
3. `PA9 -> USART1_TX`
4. `PA10 -> USART1_RX`
