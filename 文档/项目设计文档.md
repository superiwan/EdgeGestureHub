# Edge-Gesture Hub 项目设计文档

## 📋 项目概述

**项目名称**：基于边缘AI的嵌入式手持动作识别与智能交互控制器  
**英文名称**：Edge-Gesture Hub  
**开发平台**：STM32F4 + Arduino Framework  
**项目类型**：边缘AI、手势识别、人机交互

---

## 🎯 项目目标

开发一个基于边缘AI的手持式智能动作识别系统，实现以下功能：

1. **高速数据采集**：通过I2C协议采集MPU6050六轴传感器数据
2. **实时手势识别**：基于阈值算法识别8种手势动作
3. **状态可视化**：OLED实时显示识别结果和传感器数据
4. **数据调试**：UART输出至VOFA+进行波形分析
5. **AI扩展接口**：预留NanoEdge AI/STM32Cube.AI部署接口

### 功能模块说明

#### 1. 数据采集模块
- **采样频率**：20Hz（50ms周期）
- **采集数据**：
  - 加速度（X/Y/Z轴）
  - 角速度（X/Y/Z轴）
  - 温度

#### 2. 手势识别模块
- **算法类型**：基于阈值的规则识别
- **识别手势**：
  - 方向手势：Left/Right/Up/Down
  - 旋转手势：Rotate CW/CCW
  - 静止状态：Static

#### 3. 显示模块
- **显示内容**：
  - 当前手势
  - 传感器温度
  - 系统状态

#### 4. 数据输出模块
- **输出格式**：VOFA+兼容格式
- **输出频率**：20Hz
- **数据内容**：加速度、角速度、手势结果

## 🎮 手势识别算法

### 算法原理

1. **计算运动变化量**：
   ```
   delta_a = |ax - ax_prev| + |ay - ay_prev| + |az - az_prev|
   delta_g = |gx| + |gy| + |gz|
   ```

2. **手势分类逻辑**：
   - **方向手势**（delta_a > 5）：
     - Right：ax > 2
     - Left：ax < -2
     - Up：ay > 2
     - Down：ay < -2
   
   - **旋转手势**（delta_g > 100）：
     - Rotate CW：gz > 50°/s
     - Rotate CCW：gz < -50°/s
   
   - **静止状态**：
     - delta_a < 1 且 delta_g < 20

### 参数阈值

| 参数 | 阈值 | 说明 |
|------|------|------|
| 方向识别阈值 | 5 | 加速度变化量 |
| 方向判定阈值 | ±2 m/s² | X/Y轴加速度 |
| 旋转识别阈值 | 100 | 角速度变化量 |
| 旋转判定阈值 | ±50 °/s | Z轴角速度 |
| 静止判定阈值 | delta_a < 1 且 delta_g < 20 | 运动判定 |

## 🚀 应用场景

### 1. PPT翻页助手
- Left/Right手势 → 上一页/下一页
- 适用场景：演示、教学

### 2. 智能音量调节
- Rotate CW/CCW → 音量增减
- 适用场景：智能家居、音响系统

### 3. 工业安全监控
- 异常手势检测 → 紧急停机
- 适用场景：工业设备、危险环境

### 4. 游戏控制器
- 方向手势 → 游戏控制
- 适用场景：体感游戏、VR交互
